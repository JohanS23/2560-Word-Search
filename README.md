# 2560-Word-Search
Project Overview
This project implements an efficient algorithm to solve word search puzzles using concepts from data structures and algorithm design. The program reads a grid of letters and a dictionary of valid words, then identifies all words from the dictionary that can be formed within the grid.
Words can be constructed by traversing the grid in all eight possible directions (horizontal, vertical, and diagonal), and may wrap around the edges of the grid. Only words with a length of at least five characters are considered valid matches.
